# Specification: 统计视图折线图功能

## 1. 概述
在 `/summary` 统计页面中增加一个基于时间的锻炼次数折线图。该图表旨在直观展示用户过去 30 天的打卡趋势，对于没有打卡的日期，计数将自动设为 0。

## 2. 核心功能
*   **数据范围**：
    *   展示最近 30 天的俯卧撑总数。
    *   若某日无数据，折线图需将其对应的数值绘制为 0。
*   **视觉样式**：
    *   **极简线条**：采用极简设计风格，无背景网格，线条流畅。
    *   **坐标轴处理**：隐藏 Y 轴数值，仅保留 X 轴的日期标签（如：01/16）。
*   **交互细节**：
    *   **Tooltip**：点击折线上的节点时，显示该日期的具体打卡数量。
*   **移动端适配**：
    *   折线图需保持响应式，在移动端自动缩放宽度。

## 3. 技术实现
*   **图表库**：引入 `recharts`。
*   **数据获取**：
    *   扩展 `app/summary/actions.ts`，增加 `getRecentPushups(days)` 方法，获取过去 N 天的汇总数据。
    *   **后端填充**：在服务端逻辑中将缺失的日期自动填充为 0。
*   **组件**：新建 `components/pushup-chart.tsx`。

## 4. 验收标准
1.  `/summary` 页面显示最近 30 天的折线图。
2.  无打卡记录的日期在图表中正确显示为 0，而非跳过该日期。
3.  图表风格符合项目极简准则（无多余背景、隐藏 Y 轴）。
4.  在移动设备上可以正常缩放和查看。
